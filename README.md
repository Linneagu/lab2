# Portfolio website - Lab2

A contact form with JavaScript validation. This project includes validation for name, email, subject and message fields, as well as a character counter for the message textarea. 

## Setup instructions
1. Download the project - clone the repository from GitHub or download the project as a ZIP file. 
2. Open the project folder - open the downloaded project folder on your computer. 
3. Start the project - open the file index.html in any web browser.
